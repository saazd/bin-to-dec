# Binary to Decimal Converter

A simple, responsive, and modern web application that converts binary numbers into decimal format. Built with vanilla HTML, CSS (Flexbox), and JavaScript.

## Features

- **Perfect Centering**: The interface is perfectly centered on the screen using CSS Flexbox.
- **Dark Blue Theme**: A modern dark mode design inspired by Tailwind CSS colors.
- **Input Validation**: Automatically checks if the input contains only `0` and `1`.
- **Interactive UI**: Smooth button transitions and input focus animations.

## Technologies Used

- HTML5
- CSS3 (Flexbox, Variables, Transitions)
- JavaScript (Vanilla)

## How to Run

1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser.
3. Enter a binary number (e.g., `1010`) and click **Convert to Decimal**.
